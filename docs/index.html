Markdown UI

A no-class mininmalist front-end toolkit to replicate Github markdown styling

Getting Started

Installation

You can use download the stylesheet fromhere.

You can use the CDN urlhttps://unpkg.com/markdown-ui/dist/css/markdown.css

Code Template to Get started with. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="./css/markdown.css" />
    <link rel="stylesheet" href="" />
    <title></title>
  </head>
  <body></body>
</html>

Features

Markdown UI does not need any CSS class. Just include the markdown.css file in your project folder and link it the header of your html file.

Tags

Markdown UI support the following tags

<blockquotes> <code> <em> <b>
<s> <h1> <h2> <h3>
<h4> <h5> <h6> <p>
<a> <img> <hr> <ol>
<il> <li> <hr> <ol>
<table> <thead> <td> <tr>

Skills Needed

  • No CSS
  • No JavaScript
